### git flow
- main: 메인 브랜치, 현재 상태 반영
- develop: 개발을 위한 최신 버젼 브랜치
- release: 메인으로 최종 배포 전 마지막 버전 확인을 위한 브랜치
- feature: 기능별 개발 브랜치, 완료 후 develop 브랜치에 병합
- hotfix: 오류 수정이 필요시 develop 브랜치에 분기, 해결 후 병합
